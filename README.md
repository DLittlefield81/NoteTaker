## Installation
Follow these steps to install this project on your local machine.
- Open Bash editor
- mkdir NoteTaker
- cd NoteTaker
- git clone https://github.com/Dlittlefield81/NoteTaker.git
- npm i


## Usage
- After cloning this repository...
- Production mode: npm start
- Developer mode: npm run dev
